## Chrome Extension

### Page Turner
Navigate paginated webpages using your keyboard's arrow keys. Perfect for search results, image galleries and more.

### Install
https://chrome.google.com/webstore/detail/page-turner/fpbddhncmmhkaofhcnkjgcdbgcmomebo
