# Page Turner 4.2.1

A patch release with one change: Page Turner no longer requests the `tabs` permission, which it never actually needed.

## Under the hood

- Removed the `tabs` permission from the manifest. It had been requested since the very first release in 2012, but the extension only uses tabs API events (`onActivated`, `onRemoved`) to keep the toolbar icon in sync — and events never required the permission. Nothing changes in behavior; the extension just asks for less.
- Added a GPL-3.0 LICENSE file to the repository.
- Permissions are now down to `storage` plus host access — one fewer permission than 4.2.0, and none added.
